Patrones de Direcciones IPv6

Una guía rápida sobre los principales tipos de direcciones IPv6 y sus prefijos asociados.

1. Direcciones Unicast Globales (Global Unicast)

Prefijo: 2000::/3

Rango: Desde 2000:: hasta 3FFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF

Descripción: Estas son direcciones asignadas globalmente y pueden enrutar a través de Internet. Son el equivalente a las direcciones públicas en IPv4.

Ejemplo: 2001:0db8::1

2. Direcciones Unicast Locales (Unique Local)

Prefijo: FC00::/7

Descripción: Estas direcciones son para uso en redes privadas y no deben enrutarse en Internet. Equivalentes a las direcciones privadas de IPv4 como 192.168.x.x.

Ejemplo: FD00:1234::1

3. Direcciones Multicast

Prefijo: FF00::/8

Descripción: Utilizadas para enviar paquetes a múltiples destinos simultáneamente. Ideales para la transmisión de video en vivo o protocolos de descubrimiento de servicios.

Ejemplo: FF02::1 (Todos los nodos en el enlace local)

4. Direcciones Anycast

Descripción: Estas son direcciones unicast asignadas a múltiples interfaces. Los paquetes enviados a una dirección Anycast se entregan a la interfaz más cercana.

5. Dirección de Loopback

Prefijo: ::1/128

Descripción: Se utiliza para referirse al dispositivo local. Es equivalente a 127.0.0.1 en IPv4.

Ejemplo: ::1

6. Dirección No Especificada

Prefijo: ::/128

Descripción: Utilizada cuando un nodo no conoce su propia dirección. Común durante la autoconfiguración.

Ejemplo: ::

7. Direcciones IPv6 Mapeadas a IPv4

Prefijo: ::FFFF:0:0/96

Descripción: Estas direcciones permiten representar una dirección IPv4 dentro de una dirección IPv6, habilitando la interoperabilidad entre ambos protocolos.

Ejemplo: ::FFFF:192.168.1.1

8. Direcciones para Documentación

Prefijo: 2001:0DB8::/32

Descripción: Este bloque de direcciones se reserva para ejemplos y documentación técnica, no para redes reales.

Ejemplo: 2001:0DB8::1

9. Direcciones 6to4

Prefijo: 2002::/16

Descripción: Estas direcciones son utilizadas en el mecanismo de transición 6to4, permitiendo a hosts IPv4 comunicarse a través de redes IPv6.

Ejemplo: 2002:c0a8:0101::1 (Representa la dirección IPv4 192.168.1.1)

10. Direcciones Teredo

Prefijo: 2001::/32

Descripción: Utilizadas por el protocolo Teredo para permitir la comunicación IPv6 a través de una red IPv4 con NAT.

Resumen

  • FE80::/10 – Direcciones locales de enlace (Link-local)
  • 2000::/3 – Direcciones Unicast Globales (Global Unicast)
  • FC00::/7 – Direcciones locales únicas (Unique Local)
  • FF00::/8 – Direcciones Multicast
  • ::1/128 – Dirección de Loopback
  • ::/128 – Dirección no especificada
  • ::FFFF:0:0/96 – Direcciones IPv4 mapeadas en IPv6
  • 2001:0DB8::/32 – Direcciones para documentación
  • 2002::/16 – Direcciones 6to4
  • 2001::/32 – Direcciones Teredo
