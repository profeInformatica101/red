Ejercicio 2 - IPv6

Definición del problema:

Indique cuál de las siguientes expresiones de dirección IPv6 es incorrecta, y explique por qué es incorrecta:

  • a) 2001:0DB8:F181:0000:0000:0000:0000:0000
  • b) 2001:0DB8:F181:0:0:0:0:0
  • c) 2001:DB8:F181:0:0:0:0:0
  • d) 2001:DB8:F181::0:0:0:0
  • e) 2001:DB8:F181::0:0:0
  • f) 2001:DB8:F181::0:0
  • g) 2001:DB8:F181::

Respuestas y explicaciones:

Indique cuál de las siguientes expresiones de dirección IPv6 es incorrecta, explique el por qué:

  • a) 2001:0DB8:F181:0000:0000:0000:0000:0000 (Correcta)

    Esta es una dirección completa de 128 bits, con todos los grupos de 16 bits presentes, por lo tanto es válida.

  • b) 2001:0DB8:F181:0:0:0:0:0 (Correcta)

    Esta dirección está abreviada correctamente utilizando ceros, y no se usa `::` para compactar. También es válida.

  • c) 2001:DB8:F181:0:0:0:0:0 (Correcta)

    Esta es similar a la opción b), con una abreviación válida de ceros en el formato correcto.

  • d) 2001:DB8:F181::0:0:0:0 (Correcta)

    El uso de `::` está correctamente empleado para representar múltiples ceros consecutivos. Es válida.

  • e) 2001:DB8:F181::0:0:0 (Correcta)

    Similar a la opción d), el `::` se utiliza adecuadamente para compactar los ceros consecutivos. Es válida.

  • f) 2001:DB8:F181::0:0 (Correcta)

    Esta dirección usa correctamente el `::` para abreviar los ceros consecutivos y es válida.

  • g) 2001:DB8:F181:: (Incorrecta)

    Esta es incorrecta. Aunque se usa `::` para abreviar ceros consecutivos, faltan al menos 16 bits (un grupo de 16 bits) para que la dirección tenga 128 bits en total, por lo que no es válida.

Explicación General:

La opción g) es incorrecta porque no tiene el número completo de bloques que se necesitan para formar una dirección IPv6 válida. Al usar ::, se deben rellenar los ceros restantes, pero aún así la dirección debe tener 128 bits en total. En esta dirección faltan al menos 16 bits (un bloque) para que sea válida.
